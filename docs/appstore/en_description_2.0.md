## Description

IssieBoard is an accessible, fully customizable keyboard for iPad, developed by SAP Labs Israel in collaboration with the Beit Issie Shapiro Technology Consulting Center.

Designed for individuals with learning disabilities, visual impairments, motor impairments, or cognitive challenges — and for anyone learning to type — IssieBoard puts powerful personalization in the hands of caregivers, educators, and therapists.

**Setting up is simple.**
After downloading, enable IssieBoard in iPad Settings → General → Keyboard → Keyboards → Add New Keyboard. Once enabled, switch to IssieBoard in any app that uses the keyboard.

---

### Powerful Customization

- **Key colors** — set the color of keys, letters, and background across the entire keyboard
- **Zone & row styling** — apply different colors to rows, or to left/center/right sections
- **Per-key styling** — highlight individual keys with their own color
- **Focus mode** — show only the letters you choose; all others are hidden, perfect for practicing specific words
- **Font control** — choose letter size and font weight for any key group
- **Word suggestions** — intelligent word completion and next-word prediction as you type

---

### Hebrew & Arabic Support

- Full Hebrew keyboard with **nikud (diacritics)** support — tap a letter to get a popup, or use the dedicated nikud top row above the keyboard
- Arabic keyboard with harakat support

---

### Built for Real Use

- Works as a standard **iOS system keyboard** — available in any app across the entire system
- Instant live preview of every change you make in the configurator
- Auto-save as you configure — no work is lost
- Multiple keyboard profiles — create and clone keyboards for different users or use cases

---

### Who Uses IssieBoard?

IssieBoard is used by occupational therapists, special education teachers, parents, and individuals themselves. It adapts to a wide range of needs — from high contrast and large keys for visual impairments, to limited-letter layouts for early typists.

## What's New in Version 2.0 (plain text — paste into App Store Connect)

IssieBoard 2.0 is a major update with a completely redesigned configurator and powerful new features:

Redesigned app
The keyboard editor has been rebuilt from the ground up — cleaner layout, much more flexible settings.

Word suggestions
The keyboard now offers intelligent word completion and next-word prediction. Suggestions appear as you type and can be styled to match your keyboard theme.

Nikud (Hebrew diacritics)
Two ways to add vowel marks: tap a letter for a popup with all nikud options, or enable the dedicated nikud top row that sits above the keyboard for always-visible access.

Multiple keyboard configurations
Create and manage multiple keyboard configurations — each with its own layout, colors, and focus letters. Clone any profile in one tap, and share configurations with other users or devices.

Stability and performance improvements throughout.
